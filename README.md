# China Fall 2026 — Trip Dossier

Interactive route-planning site for the Fall 2026 China friends trip.
Seven route variants with photos, day-by-day itineraries, 240-hour TWOV visa clocks,
restaurant booking leads, and a decision matrix.

**Live site:** https://chriswiles.github.io/china-2026-trip/

Companion to the [trip wiki](https://chriswiles.github.io/wiles-wiki/travel/china-2026/)
(wiles-wiki repo), which holds the deep per-city pages. Photos in `images/` are from
the trip wiki vault (largely Wikimedia Commons sourced).

Maintained with Claude Code.
